HomeHunt Assistant Privacy Policy

Effective date: March 15, 2026

HomeHunt Assistant helps users track listing activity and run AI-assisted evaluations on supported listing pages.

1. Data We Handle

  • User-provided OpenAI API key (BYOK)
  • Listing activity data (viewed/contacted/evaluation metadata)
  • Listing page context used only when user triggers AI evaluation
  • User settings (model, report mode, risk priorities, retention)

2. How We Use Data

  • Track listing status and show it in extension UI
  • Run user-requested AI evaluation
  • Persist user settings and tracked state across sessions

3. Storage

Data is stored locally in the browser using chrome.storage.local.

4. Third-Party Sharing

  • We do not sell user data.
  • Data is sent to OpenAI only when user explicitly clicks AI Evaluate.

5. User Controls

  • Clear API key from Settings
  • Clear tracked data from Settings
  • Adjust retention window
